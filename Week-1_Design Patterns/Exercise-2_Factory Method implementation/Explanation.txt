This implementation employs the **Factory Method Pattern** using a single `DocumentFactory` class to generate various document types (`Word`, `PDF`, `Excel`) based on an integer input. Rather than using distinct factory classes for each document type,the `createDocument()` method accepts an integer parameter (e.g., `1` for Word, `2` for PDF) and produces the appropriate `Document` object. The factory also incorporates error handling for invalid inputs. We can validate this functionality using the `DocumentFactoryTest` class.
